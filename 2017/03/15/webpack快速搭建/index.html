webpack快速搭建

发表于 2017-03-15 | 更新于 2017-04-07 | 分类于 webpack | | 阅读次数

快速用webpack搭建项目

开始

npm需要提前安装好

项目初始化

>npm init

会自动生成 package.json

安装webpack

全局安装

>npm install webpack -g

然后安装到项目依赖中

>npm install webpack –save-dev

安装好后，就可以用webpack打包文件了

>webpack index.js index.bundle.js

index.js 要打包的文件，事先编好
index.bundle.js 要生成的打包好的文件，命其名

结构目录

创建大概的结构目录

src 源文件
  script
    coffeeScript
    typeScript
    js
  style
    less
    sass
    css
dist 静态资源文件
  js
  css

webpack配置文件

文件名

默认定义的配置文件名为
webpack.config.js

如果配置文件名称不想用这个改为 eg：webpack.dev.config.js
可在运行时webpack ---config webpack.dev.config.js

每次这样写会有点麻烦
可以在package.json中,scripts下加入

–config除此之外 还可以有

内容

webpack.config.js

entry 打包入口

如果有多个入口
entry 的值除了可以写成 字符串
还可以写成数组，对象。共有3种方式

  1. 字符串
    entry: ‘./src/script/index.js’
  2. 数组
    entry: [‘./src/script/index.js’, entry: ‘./src/script/list.js’]
  3. 对象
    entry: {
    ‘page1’: ‘./src/script/index.js’,
    ‘page2’: entry: ‘./src/script/list.js’
    }

output

output: {
filename: ‘[name].js’,
path: ‘./dist/js’
}

name指代entry中的key即
生成的两个js文件 page1.js page2.js

扩展

  • filename除了[name]变量还有 [hash]

  • output: {
    publicPath: ‘http://emloxe.github.io‘
    }
    将script中src变为绝对定位

安装html-webpack-plugin

>npm install html-webpack-plugin

webpack.config.js

webpack.config.js

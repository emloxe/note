常用快捷键

windows

  • Ctrl 多行游标

  • Ctrl+f 查找

  • Ctrl+h 替换

  • Ctrl+z 倒退

  • Ctrl+y 向前

  • Ctrl+/ 注释代码

  • Ctrl+Shift+/ 注释代码

  • Ctrl+[ 减小缩进

  • Ctrl+] 增加缩进

  • 鼠标选中文本，按下Alt+F3 一次性选择全部的相同文本进行同时编辑

  • Shift+鼠标右键 可以用鼠标进行竖向多行选择

  • Ctrl+鼠标左键 可以手动选择同时要编辑

sublime安装Package Control

发表于 2019-03-16 | 更新于 2019-03-16 | 分类于 sublime | 阅读次数

操作版本为 Version 3.2

快速安装

安装教程在这里 https://packagecontrol.io/installation

  1. 方式一
    点击 Tools -> Install Package Control

  2. 方式二
    点击 view -> Show Console

在下面的出现的输入框中输入（sublime3 版本，sublime2可在以上地址中查询）

ctrl+shift+p
在输入框中输入 install Package Control，出现相应的匹配则安装成功

安装失败

最近(2019.03.16)由于不能翻墙，导致安装失败
主要是由于 https://packagecontrol.io/installation 连接不上

可以采用如下方式
点击Preferences -> Browse Packages..
会打开文件目录


进入 C:\Users\Administrator\AppData\Roaming\Sublime Text 3\Installed Packages该目录下
下载如下链接文件，粘到Installed Packages目录
链接: https://pan.baidu.com/s/1fe-pjjwhGYLH5-Yd8vFRYQ 提取码: 3zzg

文件名称应该为Package Control.sublime-package,如果文件下载后为Packag+Control.sublime-package，则需要更改

Package Control 安装完成

按下ctrol+shift+p 输入 install Package
